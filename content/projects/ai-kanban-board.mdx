## Overview

Kanban Studio is a full-stack application that brings project management into the modern era by combining traditional Kanban boards with AI-powered assistance. Users can manage tasks through an intuitive drag-and-drop interface while leveraging natural language commands through an integrated AI chat assistant to organize their work.

## What I focused on

- Seamless integration of AI capabilities into productivity workflows, allowing users to manipulate the board through natural conversation.
- Real-time board synchronization ensuring changes made through AI operations instantly reflect in the UI.
- Building a robust full-stack architecture with clear separation between frontend and backend services.
- Session-based authentication system that maintains user state and board isolation.

## Technical details

The frontend is built with Next.js and React, providing a responsive experience with @dnd-kit for smooth drag-and-drop interactions. The backend uses FastAPI and SQLAlchemy to deliver a RESTful API with SQLite persistence. The AI integration leverages OpenRouter API, enabling natural language processing for board operations like task creation, movement, and updates. Docker and Docker Compose ensure consistent development and production environments, with deployment on Railway providing reliable hosting.
