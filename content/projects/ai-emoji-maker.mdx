## Overview

AI Emoji Maker is a full-stack product where users generate custom emoji assets with AI. The project explores monetization-friendly mechanics while maintaining a lightweight user experience.

## What I focused on

- Secure authentication and account-specific asset history.
- Credit-based generation flow to control compute consumption.
- Fast feedback loop for creating and managing outputs.

## Technical details

The stack combines React with backend services and AI model integration through Replicate. The result is a practical example of blending modern UX with AI infrastructure in a deployable web product.
