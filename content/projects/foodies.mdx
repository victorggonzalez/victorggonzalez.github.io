## Overview

Foodies is a full-stack recipe platform focused on discovery, sharing, and responsive interaction. The project demonstrates product thinking across frontend UX, backend services, and deployment.

## What I focused on

- Strong usability for browsing and contributing recipes.
- Real-time interactions to keep content fresh and engaging.
- Clean cloud deployment with fast release cycles.

## Technical details

Next.js powers the application experience, while Supabase provides backend capabilities and data services. Vercel enables smooth deployment and iteration with minimal operational overhead.
